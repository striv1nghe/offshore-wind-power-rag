## 建议检索方式

- 以每个 `## 知识片段` 下的条目作为基础 chunk。
- 检索问题如果包含“为什么误差大”“预测偏差”“功率曲线偏离”等表述，优先召回 `风速波动`、`限功率`、`偏航误差`、`桨距角` 和 `SCADA 异常`。
- 检索问题如果包含“MAE”“RMSE”“MAPE”“模型评价”等表述，优先召回 `预测误差`。
- 检索问题如果包含“SHAP”“特征贡献”“解释模型”等表述，优先召回 `SHAP 解释`。

## 典型样本标签

后续可按以下 5 类样本准备模型测试集：

1. 正常工况，预测误差小。
2. 风速波动明显，误差较大。
3. 实际功率明显偏离理论功率曲线。
4. 桨距角或转速变化明显。
5. SHAP 显示某个特征贡献很高。
