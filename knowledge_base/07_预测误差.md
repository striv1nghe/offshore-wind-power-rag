# 预测误差

## 基本定义

预测误差用于衡量模型预测功率与实际功率之间的差距。常用指标包括 MAE、RMSE 和 MAPE，不同指标对误差的敏感性不同。

在风电功率预测中，误差分析不能只看一个总指标。不同工况下的误差来源不同，例如风速波动会造成短时大误差，限功率会造成持续高估，SCADA 异常会造成不符合物理规律的误差。因此评价模型时应同时看整体指标、分工况指标、误差方向和典型样本。

## 常用指标

| 指标 | 含义 | 特点 | 注意事项 |
| --- | --- | --- | --- |
| MAE | 平均绝对误差 | 易解释，反映平均偏差 | 对大误差不如 RMSE 敏感 |
| RMSE | 均方根误差 | 对大误差敏感 | 少数异常点会明显拉高 |
| MAPE | 平均绝对百分比误差 | 表示相对误差 | 实际功率接近 0 时不稳定 |
| Bias | 平均误差方向 | 判断高估或低估 | 正负误差可能互相抵消 |
| 分位误差 | 不同误差分布位置 | 观察尾部风险 | 需要结合样本量 |

## 判断信号

- MAE 低但 RMSE 高，说明存在少数大误差样本。
- 预测值长期高于实际值，可能存在限功率、偏航误差或桨距控制。
- 预测值长期低于实际值，可能低估风速上升或功率爬升。
- MAPE 在低功率区间异常偏高时，不应单独作为结论。
- 误差集中在风速突变、额定风速附近或限功率时段，说明需要分工况分析。
- 预测曲线与实际曲线趋势相似但时间错位时，应检查时间对齐。

## 工程判断规则

### 规则 1：先看误差方向

误差方向能帮助判断问题来源。持续高估常见于限功率、偏航误差和桨距角介入；持续低估常见于风速突增、峰值低估或模型过度平滑。

### 规则 2：MAE 和 RMSE 要一起看

MAE 反映平均水平，RMSE 对大误差敏感。如果 RMSE 明显高于 MAE，说明需要找出短时大误差样本。

### 规则 3：MAPE 不适合低功率单独判断

当实际功率接近 0 时，MAPE 会被分母放大，可能出现非常高的百分比误差。低功率区间应更多参考 MAE 或归一化误差。

### 规则 4：必须分工况评价

整体误差容易掩盖问题。应按正常工况、风速波动、限功率、偏航误差、桨距角变化和 SCADA 异常分组评估。

## 与业务解释的关系

- 误差指标用于发现问题，但不能单独解释原因。
- 误差方向可以辅助判断模型高估还是低估。
- 分工况误差能说明模型在哪类场景下能力不足。
- 典型样本分析能帮助把指标转成工程语言。
- RAG 回答应同时引用指标、工况和可能原因。

## 知识片段

### 片段 1：MAE 适合观察平均误差

MAE 表示预测值与实际值之间绝对差值的平均水平。它容易解释，适合比较模型在整体样本上的平均偏差。

### 片段 2：RMSE 对大误差更敏感

RMSE 会放大较大的预测误差。如果 RMSE 明显高于 MAE，说明样本中可能存在少量误差特别大的时段，例如风速突变、限功率或 SCADA 异常。

### 片段 3：MAPE 在低功率区间要谨慎使用

MAPE 使用实际值作为分母。当实际功率接近 0 时，MAPE 可能异常放大，不能单独作为低功率或停机样本的评价依据。

### 片段 4：误差分析需要分工况

整体误差指标只能说明平均表现，不能解释错误来源。应按正常工况、风速波动、限功率、偏航误差、桨距角变化和 SCADA 异常等工况分组评估。

### 片段 5：误差方向可以辅助判断原因

如果预测值持续高于实际值，可能存在限功率、偏航误差或桨距角介入。如果预测值持续低于实际值，可能是模型低估阵风、风速输入偏低或未捕捉到快速升功率过程。

### 片段 6：大误差样本需要单独分析

少数大误差样本可能来自风速突变、限功率、SCADA 异常或时间错位。只看平均指标会掩盖这些高风险场景。

### 片段 7：正常样本误差小不代表模型稳定

模型在平稳风速和正常功率曲线样本上表现好，不代表能处理突变、限功率、偏航和控制动作。评价时必须覆盖典型难样本。

### 片段 8：误差分布比单点指标更有信息

误差分布可以显示模型是否存在系统性高估、低估或长尾大误差。相比单个平均值，误差分布更适合定位问题。

### 片段 9：低功率区间应避免过度依赖 MAPE

在停机、低风速或低功率样本中，实际功率接近 0，MAPE 可能被严重放大。此时应结合 MAE、功率归一化误差和工况标签判断。

### 片段 10：评价结论应转成工程原因

模型评价不应只说“误差大”，还应说明误差发生在哪类工况、误差方向是什么、可能由哪些工程因素造成，例如风速波动、限功率或数据异常。

## 典型问题与回答模板

### 问题 1：为什么 RMSE 很高但 MAE 还可以？

这说明整体平均误差不一定很大，但存在少数大误差样本。应检查风速突变、限功率、SCADA 异常或时间错位样本。

### 问题 2：为什么模型一直预测偏高？

持续偏高通常说明实际功率受到限制或有效风能下降。应检查限功率、偏航误差、桨距角增大、设备保护或功率测点异常。

### 问题 3：为什么 MAPE 很高？

如果实际功率接近 0，MAPE 会被放大。需要先看样本是否处于低风速、停机或低功率状态，再结合 MAE 判断误差是否真的严重。

### 问题 4：如何评价 RAG 解释是否合理？

应检查解释是否同时包含误差方向、样本工况、相关特征和工程原因。如果只引用指标而没有说明工况，解释是不完整的。
